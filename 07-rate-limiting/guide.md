# Rate Limiting for AI APIs

## Why It Matters

AI APIs are:
- Expensive per request
- Easy to abuse
- Hard to predict usage

Without limits:
→ Bots drain your wallet  
→ Attackers brute force prompts  
→ System crashes

---

## What to Limit

- Requests per second
- Tokens per minute
- Requests per user
- Requests per API key

---

## Strategy

- Per-IP limits
- Per-user limits
- Global caps

---

## Golden Rule

> If you don’t control usage, usage controls you.
