## ⚠️ What Problems This Solves

AI APIs introduce **new attack surfaces**:

- Prompt injection attacks
- Data leakage through model responses
- Unauthorized API usage
- Token abuse and cost explosions
- Sensitive data exposure in logs

This repo shows you **exactly how to defend against them**.
